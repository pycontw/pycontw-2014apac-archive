StorySense Computing 利用 Python 創造讓阿公阿嬤都愛不釋手的智慧型軟體

    StorySense Computing以人工智慧技術，利用Python開發具有語言分析及故事理解能力的應用軟體，致力於創造簡單到連阿公阿嬤都愛使用的智慧型軟體，期待能夠幫助使用者輕鬆整理大量的資料。StorySense Computing於2011年推出超實用的免費行動查號「WhatsTheNumber」App，幫助使用者在瞬間找到想去的店家或企業機關的電話，推出後好評不斷；今年StorySense Computing更擴大專業團隊編制，積極發揮Python最大的效益和功能，躍居全台最受關注的台灣新創科技公司。

 

 

 

簡單易學，隨心所欲的程式語言

    StorySense Computing創辦人Edward沈育德表示，公司在草創時期就選擇了 Python。「我們後台的演算法都是用Python寫的，因為我們很喜歡Python簡單又功能強大的特性！」StorySense Computing成功的營造出自由、歡樂，創造新價值不停歇的工作環境，是一個鼓勵創意想法的年輕團隊；成立至今，成功吸引許多渴望自我實踐的優秀人才加入。Edward談道，因為自己喜歡寫 Python，因此起初所有的開發就是用Python，所以後面進來的夥伴也理所當然的繼續寫 Python。他表示 Python 容易上手又功能強大的絕妙之處，可以用一個比喻來形容：「如果說程式語言是引擎，那麼Python就是最容易組裝起來的小跑車或是小摩托車，學習的入門門檻很低，引擎一下子就可以組裝起來，很快就可以跑很遠！其他程式語言可能像噴射飛機，功能雖強大，但組裝起來複雜難懂；相較之下，我們不假思索地選擇採用Python做開發。」StorySense Computing馬不停蹄地因應需求利用Python開發出簡單又超厲害的應用軟體，期待將台灣的高科技領域推向全世界。
